# Sales Performance Dashboard (Excel Analytics)

## Project Overview
This project is an interactive sales performance dashboard built using Microsoft Excel. It analyzes sales data to provide insights into regional performance, product categories, and customer trends.

## Objectives
- Analyze sales performance using pivot analytics
- Identify regional revenue distribution
- Provide business insights through KPIs and visualization
- Create an interactive dashboard with slicers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Dashboard design

## Key KPIs
- Total Sales
- Total Orders
- Distinct Customers (793)
- Average Order Value (AOV)

## Insights
- West region generated highest revenue
- South region underperformed, indicating growth opportunities
- Furniture category contributed highest sales
- Interactive filters allow dynamic data exploration

- Data analysis
- Dashboard creation
- Business insights
- Excel pivot analytics
